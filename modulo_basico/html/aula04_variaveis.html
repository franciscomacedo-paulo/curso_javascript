Variaveis

Tipos de Variaveis em JS

  • let
  • var
  • const
Var
A palavra-chave: var serve para declarar uma variavél no JavaScript, mais o var respeita o escopo. Ou seja mesmo que ela seja declarado dentro de uma função, ainda será possível usar ela por um comando de exibição, fora da função.

Exemplo:

            
        if(true){
            var nome = "Paulo"
        }
        console.log(nome) //resultado 'Paulo'
Let
A palavra-chave: let criado para substituirr o var, essa sim respeita o escopo, ela funciona apenas no local onde ela foi declarada.

Exemplo:

            
        if(true){
            let nome = "Paulo"
        }
        console.log(nome) //resultado 'referenceError'
Const
A palavra-chave: const é usado para declarar valores que não podem mudar ou alterar o seu valor.

Exemplo:

            
        const curso = "JavaScript"
        curso = "Python"
        console.log(curso) //resultado 'TypeError: Assignment to constant variable'
