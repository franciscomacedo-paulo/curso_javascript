Trocar de Abas

HTML
HTML é a linguagem padrão para a construção das páginas web, que significa, (HyperText Marckup Language) ou seja Linguagem de Marcação de Texto, é com o HTMl que se cria a estrutura da texto.
CSS
CSS é a linguagem responsável pelo estilo de uma página web, é ele quem deixa as coisas mais agradaveis de se ver, significa (Cascading Style Sheets), ou seja Folhas de Estilos em Cascata.
JavaScript
JavaScript é a linguagem pela iteractividade da página, ou seja ele da vida, as páginas webs, tornandos mais dinamicas controlando assim o que acontece, em cada click.
React
O React é uma biblioteca JavaScript usada para criar interfaces de usuário (UI) — principalmente para aplicações web modernas e interativas. Ele foi criado pelo Meta (antigo Facebook) e hoje é uma das tecnologias mais usadas no mundo front-end.
Nodejs
O Node.js não é uma linguagem de programação, mas sim um ambiente de execução (runtime) que permite rodar JavaScript fora do navegador, ou seja, no servidor. Antes do Node.js, o JavaScript era usado quase exclusivamente para dar interatividade a sites no lado do cliente. Com ele, você pode criar aplicações completas utilizando a mesma linguagem na frente (frontend) e atrás (backend).
